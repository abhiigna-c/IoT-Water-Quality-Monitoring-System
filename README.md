
# IoT Water Quality Monitoring System

## 📌 Project Overview
This project is designed to monitor water quality in real-time using IoT technology. It measures parameters like Total Dissolved Solids (TDS) and temperature and displays the data on cloud platforms and mobile applications.

---

## ⚙️ Components Used
- NodeMCU (ESP8266)
- TDS Sensor
- Temperature Sensor
- LCD Display
- Buzzer
- Wi-Fi Module

---

## 🚀 Features
- Real-time water quality monitoring
- Cloud data visualization using ThingSpeak
- Remote monitoring using Blynk mobile app
- Threshold-based alert system
- LCD display for local monitoring

---

## ☁️ Platforms Used
- ThingSpeak (Data visualization)
- Blynk (Mobile monitoring)

---

## 🛠️ Working
1. Sensors collect water quality data
2. NodeMCU processes the data
3. Data is sent to cloud platforms via Wi-Fi
4. Users can monitor data remotely
5. Alerts are triggered if values exceed safe limits

---

## 📊 Results
- Real-time data successfully displayed on ThingSpeak
- Mobile monitoring achieved using Blynk
- System responds to unsafe conditions with alerts
  
---

## 📌 Conclusion
The system provides an efficient and low-cost solution for monitoring water quality in real-time and can be used in homes and environmental applications.
